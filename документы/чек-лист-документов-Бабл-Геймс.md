# Чек-лист документов ООО «Бабл Геймс» — от регистрации до продаж на WB/Ozon

> ✅ — уже готово (проекты в этой папке). Остальное — по этапам, в порядке необходимости.

## Этап 1. Регистрация (пакет в исполком)

- ✅ Устав (2 экз. + электронная копия .doc/.rtf)
- ✅ Протокол № 1 учредительного собрания
- ✅ Корпоративный договор (подписать одновременно, в исполком не подаётся)
- [ ] Справка о согласовании наименования (портал ЕГР, онлайн, до 1 раб. дня)
- [ ] Гарантийное письмо арендодателя о предоставлении юрадреса (нежилое помещение)
- [ ] Заявление о государственной регистрации (форма исполкома / портала ЕГР)
- [ ] Квитанция об уплате госпошлины 1 БВ (при онлайн-подаче — не нужна)
- [ ] Паспорта учредителей

## Этап 2. Первая неделя после регистрации

- [ ] Контракт (трудовой договор) с директором — подписывает второй участник
- [ ] Приказ № 1: вступление директора в должность, возложение обязанностей по бухучёту (или назначение ответственного)
- [ ] Договор с бухгалтерской компанией (аутсорс)
- [ ] **Уведомление о переходе на УСН** — жёсткий дедлайн 20 рабочих дней с даты регистрации (если выбрали УСН; решение — с бухгалтером)
- [ ] Договор аренды юрадреса/офиса + акт приёма-передачи
- [ ] ЭЦП (ГосСУОК) на директора
- [ ] Открытие расчётного счёта BYN + валютных (RUB — обязательно под выплаты маркетплейсов)
- [ ] Книга учёта проверок + книга замечаний и предложений
- [ ] Учётная политика (приказом, до первой операции)

## Этап 3. Интеллектуальная собственность (до передачи игры в производство!)

- [ ] Договоры **отчуждения исключительных прав** от каждого учредителя обществу — на всё придуманное до регистрации (правила, название игры, механики, черновики)
- [ ] Шаблон договора **авторского заказа** с отчуждением прав — для художников, иллюстраторов, авторов текстов, дизайнеров (каждый подрядчик подписывает ДО начала работы)
- [ ] Шаблон **NDA** для подрядчиков и типографий
- [ ] Проверка бренда игр по базам **НЦИС (РБ) и Роспатента (РФ)** — классы МКТУ 28 (игры), 16 (печатная продукция), 41; особо проверить конфликт с знаками «BUBBLE» (Bubble Comics, РФ)
- [ ] Заявка на **товарный знак**: НЦИС (РБ) + Роспатент (РФ — рынок сбыта!); подать до выхода на маркетплейсы
- [ ] Лицензии на шрифты и стоковые изображения, используемые в макетах (частая дыра: шрифт с «пиратской» лицензией = блокировка карточки по жалобе)
- [ ] Депонирование правил и макетов игры (необязательно, но дёшево и полезно как доказательство авторства)

## Этап 4. Продукт и производство

- [ ] Договор с типографией/производителем компонентов: контрактное производство, NDA, права на макеты и печатные формы — у общества, минимальные партии, контроль качества образцов
- [ ] **Сертификация**: настольные игры для детей до 14 лет попадают под ТР ТС 008/2011 «О безопасности игрушек» → обязательный сертификат соответствия ЕАЭС (протоколы испытаний в аккредитованной лаборатории). Игры «14+/18+» — под ТР не попадают → оформить **отказное письмо** (письмо о неотнесении к объектам техрегулирования). Возрастную маркировку выбирать осознанно: «12+» на коробке = нужен сертификат
- [ ] Этикетка/маркировка по требованиям ЕАЭС: наименование, изготовитель и его адрес, «изготовлено по заказу ООО „Бабл Геймс“», страна происхождения, возраст, дата изготовления, знак EAC (для сертифицированных)
- [ ] Штрихкоды: регистрация в GS1 Belarus (или через сервисы маркетплейсов)
- [ ] Паспорт качества / декларация изготовителя на партию (внутренний документ)

## Этап 5. Выход на Wildberries / Ozon

- [ ] Регистрация продавца: WB Partners и Ozon Seller на белорусское ООО (УНП, счёт в RUB, директор с ЭЦП; оферта принимается в личном кабинете)
- [ ] Договоры-оферты маркетплейсов — сохранить актуальные версии, показать бухгалтеру (момент признания выручки, отчёты агента)
- [ ] Валютный контроль: регистрация сделки в банке (поступления RUB от WB/Ozon), консультация банка по порогам
- [ ] Документы на товар для маркетплейса: сертификат/отказное письмо, товарный знак (или заявка), карточки товара
- [ ] ЭДО с Ozon (для схемы FBO поставок в РФ) — уточнить текущие требования к продавцам из ЕАЭС
- [ ] Логистика в РФ: договор с фулфилментом/перевозчиком, СМР-накладные; ЕАЭС = без таможни, но статистическое декларирование в РБ при вывозе — уточнить у бухгалтера порог
- [ ] Политика обработки персональных данных + согласия (если будет свой лендинг/предзаказы)

## Этап 6. Кадры (по мере найма)

- [ ] Штатное расписание, ПВТР (правила внутреннего трудового распорядка)
- [ ] Шаблон контракта работника + должностные инструкции
- [ ] Приказы о приёме, уведомления в ФСЗН
- [ ] Обязательство о неразглашении коммерческой тайны + положение о коммерческой тайне

## Порядок действий (сводно)

1. Название → юрадрес → подписание корпдоговора → регистрация (этапы 1–2, ~2 недели).
2. Параллельно: ИС-пакет (этап 3) — до того, как художник нарисует первую карту.
3. Прототип → типография → сертификация/отказное (этап 4, закладывать 3–6 недель на испытания).
4. Кабинеты WB/Ozon, валютный контроль, первая поставка (этап 5).

*Проекты договоров этапов 3–4 (отчуждение прав, авторский заказ, NDA, договор с типографией) могу подготовить следующим шагом.*
